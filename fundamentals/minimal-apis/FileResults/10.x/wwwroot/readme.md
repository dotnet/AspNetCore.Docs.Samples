# ASP.NET Core File Results Demo

This markdown file demonstrates serving different file types.

## Features Demonstrated

- **TypedResults.File()** - Return file content from byte arrays or streams
- **TypedResults.PhysicalFile()** - Return files from physical disk paths
- **TypedResults.VirtualFile()** - Return files from web root
- **TypedResults.Bytes()** - Return raw byte content

## Benefits of TypedResults

1. **Type Safety** - Compile-time checking of return types
2. **OpenAPI Support** - Automatic schema generation for API documentation
3. **Testability** - Strongly-typed results are easier to unit test
